Sistemas operativos

1. Qué es un Sistema Operativo

Es un software que administra los recursos de hardware y software de una computadora y proporciona servicios para los programas.

2. Componentes de un Sistema Operativo

Núcleo (Kernel), interfaz de usuario, sistema de archivos, controladores de dispositivos.

3. Tareas del Sistema Operativo

Administración de procesos, memoria, archivos, y dispositivos; seguridad; interfaz de usuario.

4. Funciones de un Sistema Operativo

Gestionar recursos, ejecutar programas, controlar dispositivos periféricos, y proporcionar una interfaz de usuario.

5. Características de un Sistema Operativo

Multitarea, multiusuario, seguridad, estabilidad, y soporte para redes.

6. Tipos de Sistema Operativo

Monousuario, multiusuario, monotarea, multitarea, tiempo real, distribuido, y embebido.

7. Sistemas Operativos Para PC

Windows, macOS, Linux.

8. Sistemas Operativos Móviles

Android, iOS.

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

PCs (Windows, Linux), servidores (Windows Server, Unix), dispositivos móviles (Android, iOS).

10. Compatibilidad de los Sistema Operativo

Capacidad de ejecutar aplicaciones diseñadas para versiones anteriores o diferentes sistemas operativos.

11. Cuál es la función básica de un Sistema Operativo

Administrar los recursos de hardware y software y proporcionar servicios para la ejecución de programas.

12. Mencione tres Sistema Operativo más utilizados

Windows, macOS, Linux.

13. Menciona algunas tareas que realizan los Sistemas Operativos

Gestión de memoria, ejecución de programas, control de periféricos, manejo de archivos.

14. Qué es un sistema operativo multitareas

Permite ejecutar múltiples tareas o procesos simultáneamente.

15. Qué es un sistema operativo multiusuario

Permite que varios usuarios utilicen el sistema de manera concurrente.

16. Cómo funciona un sistema operativo

Gestiona recursos hardware y software, proporciona servicios para aplicaciones, y actúa como intermediario entre el usuario y el hardware.

17. Cómo se utiliza un sistema operativo en un entorno gráfico

A través de una interfaz gráfica de usuario (GUI) que permite interactuar con el sistema mediante ventanas, iconos, y menús.

18. Cuándo inicia el funcionamiento del Sistema Operativo

Al encender la computadora, durante el proceso de arranque (boot).

19. Mencione las dos interfaces que forman parte del Sistema Operativo

Operativo: Interfaz de Línea de Comandos (CLI) y la Interfaz Gráfica de Usuario (GUI).

20. Qué es un proceso en entorno de los sistemas operativos

Es una instancia de un programa en ejecución que el sistema operativo gestiona.

21. Qué es un sistema operativo distribuido y centralizado

Distribuido: administra un grupo de computadoras conectadas; Centralizado: controla los recursos y aplicaciones de un único sistema.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

NTFS, FAT32.

23. Mencione otras de las características actuales de los sistemas operativos

Virtualización, soporte para arquitecturas multiprocesador, seguridad avanzada, y gestión de energía.

24. Que permite la exigencia de muchos controladores en un sistema operativo

Permite la comunicación y el control de diversos dispositivos de hardware, facilitando la compatibilidad y funcionalidad.
